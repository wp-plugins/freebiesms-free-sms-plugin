FreebieSMS Free SMS plugin for wordpress that earns you money as people send SMS from your site. We pay 50p ($1) for every SMS received.


== Description ==

Thanks to Martin Fitzpatrick at MuTube we've now got a Free SMS plugin for Wordpress, making it easy to send SMS messages from your wordpress page.

Personally, I found the default theme not to suit the plugin, so I
changed theme to Vigilance 1.47 by The Theme Foundry 

To change the affiliate ID in the wordpress admin section go to
'Settings' and there should be a tab in there for 'FreebieSMS' click
on that and the affiliate number is in there.

Please visit our website at www.freebiesms.co.uk, or visit our blog at
freebiesms.blogspot.com



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= How do I get my Affiliate ID? =

Go to www.freebiesms.co.uk, click on affiliates, select the link option,
press continue, enter your details, press "Generate HTML".

In the Link you will see a number. This is your affiliate ID.
